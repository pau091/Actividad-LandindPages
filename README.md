# 🎨 15 Landing Pages — Portafolio Estudiantil

Colección de **15 landing pages** diseñadas y desarrolladas como ejercicios prácticos de maquetación web con HTML y CSS.

---

## 🚀 Cómo usar

1. **Descomprime** el archivo `15LandingConde.zip` (si aún no lo has hecho).
2. Abre el archivo **[`index.html`](./index.html)** en tu navegador.
3. Desde ahí podrás previsualizar o abrir cualquiera de las 15 landing pages.

> ⚠️ Para que las previsualizaciones funcionen correctamente, todos los archivos deben estar **en la misma carpeta** sin mover ni renombrar nada.

## 🛠 Tecnologías utilizadas

- **HTML5** — Estructura semántica de cada página
- **CSS3** — Estilos, layouts (Flexbox/Grid) y animaciones
- **Imágenes** — Recursos visuales propios o de referencia (carpeta `img/`)

---

## 👁 Galería interactiva

El archivo `index.html` incluye:

- **Vista previa en miniatura** de cada landing page (iframe embebido)
- **Modal a pantalla completa** para explorar cada página sin salir de la galería
- **Navegación con teclado**: `←` `→` para cambiar de página, `Esc` para cerrar
- **Buscador** para filtrar por nombre o temática
- **Botón "Abrir"** para ver cada landing en su propia pestaña

---

## 📝 Notas para el profesor

- Cada carpeta (`pageXX`) es independiente y contiene su propio HTML, CSS e imágenes.
- Algunas páginas pueden estar incompletas o en proceso (ej: `page15`).
- Se recomienda abrir desde un servidor local (ej: Live Server en VS Code) para evitar restricciones de iframes en algunos navegadores.
